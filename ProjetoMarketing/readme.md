
# Projeto Marketing com LLMs e IA Generativa

(English version below)

## Descrição

Este projeto foi desenvolvido durante as aulas de um curso sobre Inteligência Artificial Generativa e Agentes, com finalidade educacional e de aprendizado prático. A proposta é explorar o uso de modelos de linguagem (LLMs), técnicas de prompt engineering e integração com ferramentas modernas para criar experiências relacionadas a marketing digital e geração de conteúdo.

O repositório concentra-se em um notebook com exemplos práticos de aplicação de IA Generativa em cenários de marketing, incluindo experimentos com modelos via APIs e a construção de uma interface simples com Streamlit.

## Objetivo

O objetivo principal deste projeto é aplicar os conceitos apresentados ao longo do curso, como:

- uso de LLMs para geração de texto;
- integração com bibliotecas como LangChain;
- exploração de modelos proprietários e open source;
- construção de fluxos de interação com prompts e cadeias de processamento;
- adaptação do protótipo para uma interface mais prática com Streamlit.

## Tecnologias utilizadas

As tecnologias e bibliotecas observadas no projeto incluem:

- Python
- Jupyter Notebook
- LangChain
- LangChain Groq
- LangChain OpenAI
- OpenAI Python SDK
- Streamlit
- python-dotenv
- ipywidgets
- Hugging Face Transformers (em partes do notebook)

> Algumas dependências podem ter sido introduzidas ao longo do desenvolvimento do notebook e podem variar conforme a execução realizada.

## Estrutura do projeto

Atualmente, o repositório contém:

- [1 - LLMs para empresas e negócios - Marketing.ipynb](1%20-%20LLMs%20para%20empresas%20e%20neg%C3%B3cios%20-%20Marketing.ipynb): notebook principal com o desenvolvimento do projeto.

## Como executar o projeto

1. Abra o notebook em um ambiente Jupyter ou no Google Colab.
2. Instale as dependências indicadas nas células iniciais do notebook.
3. Configure as chaves de API necessárias para os modelos utilizados.
4. Execute as células em ordem para reproduzir os exemplos e experimentos.
5. Para a versão baseada em Streamlit, execute o script gerado a partir do notebook com o comando:

```bash
streamlit run app.py
```

> O projeto foi desenvolvido como material de estudo, portanto a execução pode depender de credenciais de API, disponibilidade de rede e recursos computacionais.

## Funcionalidades

O notebook aborda, de forma prática, funcionalidades como:

- geração de textos para marketing digital;
- uso de prompts com foco em SEO e escrita persuasiva;
- integração com modelos de linguagem via APIs;
- construção de cadeias de processamento com LangChain;
- experimentação com modelos públicos e proprietários;
- adaptação para uma interface interativa com Streamlit.

## Resultados ou exemplos de uso

Os exemplos presentes no projeto incluem:

- geração de textos para campanhas e conteúdos de marketing;
- criação de prompts direcionados a objetivos específicos, como persuasão, engajamento e clareza;
- teste de diferentes abordagens de interação com modelos de linguagem;
- validação de uma aplicação simples convertida para execução em Streamlit.

Os resultados variam conforme o modelo escolhido, a configuração do prompt e os dados fornecidos durante a execução.

## Aprendizados

Este projeto oferece uma oportunidade prática para compreender:

- como funcionam os principais fluxos de uso de LLMs;
- como estruturar prompts de forma mais eficaz;
- como integrar modelos de linguagem em aplicações reais;
- como adaptar protótipos de laboratório para uma interface mais acessível;
- como pensar criticamente sobre uso de IA Generativa em contextos práticos e educacionais.

# Marketing Project with LLMs and Generative AI

## Description

This project was developed during classes of a course on Generative Artificial Intelligence and Agents, for educational and practical learning purposes. The goal is to explore the use of large language models (LLMs), prompt engineering techniques, and integration with modern tools to create experiences related to digital marketing and content generation.

The repository focuses on a notebook with practical examples of applying Generative AI in marketing scenarios, including experiments with models through APIs and the creation of a simple interface with Streamlit.

## Objective

The main objective of this project is to apply the concepts presented throughout the course, such as:

- using LLMs for text generation;
- integrating libraries such as LangChain;
- exploring proprietary and open-source models;
- building prompt-based interaction flows and processing chains;
- adapting the prototype to a more practical interface with Streamlit.

- Jupyter Notebook
- LangChain
- LangChain Groq
- LangChain OpenAI
